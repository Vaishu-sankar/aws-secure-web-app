#  Enterprise Cloud-Based Web Application on AWS 

##  Overview

This project demonstrates deploying a secure, scalable, and highly available web application using AWS Free Tier services.

##  Services Used

* EC2 (Compute)
* VPC (Networking)
* S3 (Storage)
* RDS (Database)
* IAM (Security)
* Load Balancer
* CloudWatch (Monitoring)

##  Web Server

Apache (httpd)

##  Objective

* Deploy web server on AWS
* Implement high availability
* Use secure architecture
* Document all steps

##  Outcome

Successfully deployed Apache-based web application in AWS .
